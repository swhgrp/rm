Setup Instructions
Getting your Clover credentials:
  1. Log in to your Clover Dashboard
  2. Go to Account & Setup → API Tokens
  3. Create a new API Token with these permissions:
    • Read Orders
    • Read Inventory
  4. Copy your Merchant ID and Access Token
  5. Paste them in the form and click "Test Connection"
Security Note: Your API credentials are stored securely and encrypted. Never share your access token with anyone.
